# ESP32-Bluetooth-A2DP-Receiver

ESP32 based Bluetooth A2DP receiver

### Parts list:
* ESP32 development board
* PCM5102A
* RGB LED

### FAQ / Troubleshooting
* My board (**ESP32-D0WD-V3**) doesn't appeared in boards list in Arduino IDE: select **AI Thinker ESP32-CAM**
